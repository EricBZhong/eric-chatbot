Full Disk Access Required

Your Mac needs to give this app permission to read iMessage data. Here's how:

  1. Open System Settings (click the Apple menu → System Settings)
  2. Go to Privacy & Security → Full Disk Access
  3. Click the + button and add Terminal (in Applications → Utilities)
  4. Toggle it ON
  5. Close and re-open the app, then try importing again

If you launched from start.command, add "Terminal.app". If you use iTerm, add "iTerm.app" instead.
